# Frequently Asked Questions

### How can I get whitelisted on a server?

By joining the Flux Discord server you have to go through a registering process, after which you can select a server from the #servers channel and then requesting whitelist from the respective server #info channels.

### Where is the server located?

Chicago, Illinois, USA

### Are servers backed up?

All of our servers are backed up every 3-6 hours.

### Can I buy a rank or donate?

You can go to our store [here](https://store.fluxsmp.com) to do both. 

### I have a suggestion for Flux MC. Where can I put it?

If you have a suggestion for Flux MC, please join our discord and go to the #suggestions channel.

### How can I join staff on Flux MC?

If you think you can bring something valuable to the team please message @Pixel#8020 on discord and we can discuss a possible position for you. Only requirement is that you are 16+ and you make an effort to be around and help.

### What hardware does the server run on?

- Location: Chicago, Illinois, USA 
- CPU: Ryzen 7 5800x
- RAM: 128GB DDR4
- Storage: Dual 2TB NVMe SSDs in RAID 1
