### Welcome to the Flux MC Wiki

This wiki is a place where you can find information about all things Flux MC. If you have any questions, feel free to ask in the [Discord](https://discord.gg/FtZkmWcNcB). 

### How to Navigate
The sidebar on the left contains all the pages on the wiki. You can also use the search bar to find a specific page or topic. 

### How to Contribute
If you would like to contribute to the wiki, you can do so by either suggesting topics to be included in the discord. Or you can make a pull request on the [GitHub Page](https://github.com/Flux-MC/Wiki). This wiki is built using [Docusaurus](https://docusaurus.io/) and is very easy to contribute to.
